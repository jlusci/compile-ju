A pretty (functional) web interface for the JU Programming Language.

Would you like to say hello? Play some fizzbuzz? Create an address book? Go ahead, give it a try!

Basic syntax rules for the JU language:

  • Leading white space is irrelevant
  • Use semicolons to end expressions
  • To define functions: function main(){ [block] }
  • To declare variables: var y = 2;
    • (in place of 2 you can also define strings, ints, floats, math expressions, lists, and dictionaries)
  • To reassign variables: y = y + 1;
  • To call functions previously defined with parameters: new_fun(2);
  • To create if statements: if ( [expression] ) { [block] } else { [block] }
  • To create for loops: for([var_def]; [expression]; [var_assign] ) { [block] }
  • To create while loops: while( [expression] ) { [block] }
  • To print: print [expression];
